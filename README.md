# Mangan-Apps
Web sederhana yang berisi daftar restoran dan sebagai syarat terakhir untuk submission kelas Dicoding.

Memiliki fitur:  

- menyukai restoran 
- mencari restoran favorit
- dapat di-install pada PC maupun smartphone (mekanisme penerapan PWA)
- link Mangan Apps dapat diakses di <a href="https://mangan-apps.netlify.app/" target="_blank">sini</a>
